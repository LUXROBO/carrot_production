
## 개요
캐롯 플러그 양산 과장의 품질 검증(QA) 프로그램입니다. 
이 프로그램은 캐롯 플러그 제품의 품질을 보장하고 생산 공정에서의 불량률을 최소화하기 위해 개발되었습니다.

### 지원 단말 종류
1) LUX1(SARA-R4)
- Branch: R4_QA


2) LUX2 (BG770)
- Branch: BG770_QA


3) EMT
- Branch:  EMT_QA


### 기능
- 단말기별 QA 테스트 수행
- 자동화된 품질 검증 프로세스
- 실시간 테스트 결과 모니터링
- 불량 사항 자동 감지 및 리포팅
- 생산 데이터 분석 및 품질 통계
